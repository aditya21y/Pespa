# Pespa Mobile Apps
Pespa (prediksi stunting pada anak) merupakan aplikasi mobile yang dibuat untuk melakukan prediksi kondisi stunting pada anak dengan fokus khusus pada Kecamatan Pitu, Kabupaten Ngawi.
